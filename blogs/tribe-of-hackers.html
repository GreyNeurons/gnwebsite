Keytake aways from Tribe of Hackers

Tribe of Hackers is a collection of interviews with 70 cyber security experts. Interview questions asked to these cyber security experts is standard. Questions cover wide topics related to cyber security from cybersecurity tools/technolgies, processes to making careers in cybersecurity.

if you had to jot down the most common advise given by these 70 odd experts, it would really come down to following

For Individuals, you need to do following

  • Use strong password ( min 16 digits, alphanumeric + special characters ).
  • Do not repeat passwords across sites.
  • Use password manager ( it takes care of both of the above items ).
  • Use VPN.

    • For Organizations, resoundingly the advice is

  • Focus on people then tools ( Training budget / Tools budget may be a good metrics to track )

    • Obviosly insane focus on buying more tools , technologies is not helping any one because tools do end up creating tons and tons of data with very minimal ways to auto remediate the situation. Tool implementation eats up so much budget which results in reduced budget for staffing and training.
